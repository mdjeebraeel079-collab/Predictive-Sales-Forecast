# Predictive Sales Forecast Dashboard

## Project Overview

This project focuses on analyzing historical sales data and forecasting future sales trends using Microsoft Excel.

The dashboard provides insights into sales performance, profit analysis, sales targets, regional performance, and future sales forecasting.

## Objectives

- Analyze historical sales performance
- Compare actual sales with targets
- Forecast future sales trends
- Identify high-performing regions
- Generate business insights

## Tools Used

- Microsoft Excel
- Pivot Tables
- Charts
- Forecast Sheet
- Data Visualization

## Dashboard KPIs

- Total Sales
- Total Profit
- Total Customers
- Average Monthly Sales

## Dashboard Features

### Monthly Sales Trend
Tracks monthly sales growth over time.

### Actual Sales vs Target Sales
Compares achieved sales against planned targets.

### Sales Forecast
Predicts future sales for the next 6 months.

### Profit by Category
Shows profit contribution from each product category.

### Sales by Region
Compares sales performance across regions.

## Key Insights

- Sales show a strong upward trend.
- Electronics category contributes the highest profit.
- South region generates the highest sales.
- Forecast indicates continued sales growth.
- Actual sales are approaching sales targets.
